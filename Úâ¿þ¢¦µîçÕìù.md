## 🚀 快速部署指南

### 方式一：首次部署（完整上传）

1. 上传 `xuanxiang-php-v4.0-final.zip` 到服务器
2. 解压到网站根目录
3. 设置权限：`chmod 755 data`
4. 访问后台配置API

### 方式二：更新部署（保留数据）

**使用自动更新脚本**（推荐）

1. 上传 `xuanxiang-php-v4.0-final.zip` 到网站根目录
2. 在SSH中运行：
```bash
bash update.sh
```

脚本会自动：
- ✅ 备份现有数据
- ✅ 更新所有代码文件
- ✅ 保留 data 目录（卡密、订单不丢失）
- ✅ 恢复权限

**手动更新（如果没有SSH）**

1. 在小皮面板中，先备份 `data` 目录
2. 删除所有文件（除了 data）
3. 上传解压新版本
4. 还原 data 目录

### 🔧 常见问题

**1. API测试失败**
- 检查后台"系统配置"中的API密钥是否正确
- 确认API地址格式正确（如：https://api.openai.com/v1/chat/completions）

**2. 更新后卡密消失**
- 使用 `update.sh` 脚本会自动保留数据
- 手动更新前务必备份 `data` 目录

**3. 权限错误**
```bash
chmod 755 data
chmod 644 data/*.json
```
