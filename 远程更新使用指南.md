# 🚀 玄象远程更新工具使用指南

## ✨ 功能说明

不用再手动删除上传了！本地一键推送更新到服务器。

---

## 📦 首次部署

### 第1步：上传完整包
上传 `xuanxiang-php-v4.0-final.zip` 到服务器并解压

### 第2步：确保 update.php 存在
检查网站根目录是否有 `update.php` 文件

### 第3步：修改更新密钥（可选）
编辑 `update.php` 第4行，修改密钥：
```php
define('UPDATE_KEY', '你的密钥');
```

---

## 🎯 日常更新流程

### 方式1：使用网页工具（推荐）

1. **打开更新工具**
   - 双击 `一键更新工具.html`

2. **选择文件**
   - 点击"选择 ZIP 文件"
   - 选择 `xuanxiang-php-v4.0-final.zip`

3. **配置服务器**
   - 服务器地址：`http://ai.olbaba.com`
   - 更新密钥：`xuanxiang2026`（或你修改的密钥）

4. **点击更新**
   - 点击"开始更新"按钮
   - 等待完成（10-30秒）

5. **完成！**
   - 看到"更新成功"即可访问网站

---

## 🔍 更新工具原理

1. **上传**：ZIP包通过HTTP POST上传到 `update.php`
2. **备份**：自动备份 `data` 目录到 `data_backup_时间戳`
3. **解压**：临时目录解压新文件
4. **更新**：删除旧代码，复制新代码（保留 `data`）
5. **清理**：删除临时文件，设置权限

---

## ⚠️ 注意事项

1. **数据安全**
   - 每次更新自动备份 `data` 目录
   - 卡密、订单、用户数据不会丢失

2. **网络要求**
   - 需要能访问服务器（本地或VPN）
   - 支持跨域上传

3. **安全建议**
   - 修改默认更新密钥
   - 更新完成后可以删除 `update.php`（下次更新再上传）

## 🆘 故障排查

**问题：密钥错误**
- 检查 `update.php` 中的密钥是否正确
- 更新工具中输入的密钥是否一致

**问题：上传失败**
- 检查服务器PHP配置：`upload_max_filesize` 和 `post_max_size`
- 建议设置为 50M 以上

**问题：权限错误**
- 服务器上执行：`chmod 755 data`
